Python Version 2.1.1, May 27, 2006

2.1.1 is the second full release of the Python version of StringTemplate. It is functionally equivalent to the Java StringTemplate Version 2.1.1. A patch for bug in the ASTExpr module provided by Brad Knotwell has been applied to this release. A test case for the bug has now been added to the unit tests.
